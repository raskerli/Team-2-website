Website Project

Welcome to our website project! This project is created as part of our learning journey on CodeAcademy. we are excited to show our skills in HTML and CSS to build a website.

Information about HTML course

HTML course provides a hands-on learning experience, allowing students to practice coding in a browser-based environment. Students build projects progressively, starting with basic HTML elements and advancing to more complex structures. The interactive nature of the platform helps reinforce learning through practical application.

Information about CSS course

CSS course follows a similar interactive approach. Students practice CSS styling directly in the browser. As students progress, they apply CSS to enhance the appearance of their web pages, gaining practical skills. The immediate feedback loop helps students understand how CSS affects the visual presentation of HTML elements.

Information about JavaScript course

JavaScript course is known for its interactive coding exercises that allow students to practice JavaScript in a web-based environment. Students work on progressively complex projects, applying JavaScript concepts to real-world scenarios. Codecademy provides instant feedback on code, helping students understand and correct errors quickly.
